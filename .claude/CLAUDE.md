# Project Instructions

## CRITICAL: Load Lessons Before Any Task

**Before starting ANY task, instruction, or responding to ANY user message:**

1. **READ** `.claude/task/lessons.md` first
2. **THINK**: Does the user's current message contain feedback (direct or indirect)?
3. **UPDATE**: If yes, add the lesson to `lessons.md` immediately
4. **APPLY**: Follow all lessons to avoid repeating mistakes

This is a learning system. Every interaction is an opportunity to capture feedback and improve.

---

## Lesson Update Protocol

For EVERY user message, ask yourself:
- Is this a correction? → Add to lessons.md
- Is the user frustrated about something I did? → Add to lessons.md  
- Did I do something the user didn't want? → Add to lessons.md
- Is there implicit feedback in their tone/words? → Add to lessons.md
- Can this inform future work? → Add to lessons.md

**Format for new lessons:**
```markdown
### [Category]
- **[Short title]**: [What to do/not do and why]
```
